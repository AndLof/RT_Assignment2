Description\+: ~\newline
 This node consists to an action client that allows the user to control the robot into the simulation environment, providing a simple interface to set the desired goal position or to cancel it; besides, it publishes real time informations about robot\textquotesingle{}s velocity and position. Whenever a goal position is reached or canceled, a message is diplayed on the terminal.
